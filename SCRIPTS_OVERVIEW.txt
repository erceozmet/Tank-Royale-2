Tank Royale 2 - Scripts Overview
========================================

üöÄ RECOMMENDED: ONE COMMAND TO START EVERYTHING
===============================================
./start-everything.sh  - Start all containers + Go servers
./stop-everything.sh   - Stop everything

üéØ MAIN LAUNCHER (tank.sh)
==========================
./tank.sh setup       - First-time setup (creates all containers)
./tank.sh start       - Start all containers
./tank.sh stop        - Stop all containers
./tank.sh databases   - Start databases only
./tank.sh monitoring  - Start monitoring only
./tank.sh go-servers  - Start Go servers only
./tank.sh go-stop     - Stop Go servers only
./tank.sh all         - Start everything
./tank.sh stop-all    - Stop everything
./tank.sh help        - Show all commands

üéÆ GO SERVER MANAGEMENT
=======================
./start-go-servers.sh - Start API + Game servers
./stop-go-servers.sh  - Stop Go servers

üì¶ INDIVIDUAL SCRIPTS (in ./scripts/)
======================================
setup-podman.sh       - First-time setup (creates everything)
start-all.sh          - Start all existing containers
stop-all.sh           - Stop all containers (keeps data)
start-databases.sh    - Start databases only
start-monitoring.sh   - Start monitoring stack only
commands.sh           - Display all available commands
